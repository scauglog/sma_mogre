Notice d'installation

ouvrir le fichier solution visual studio "tutorial.sln" avec visual studio
Exécuter la solution (F5)
Dans la fenêtre qui apparait choisir "direct3D9 Rendering Subsystem" dans le menu déroulant
Régler les options dans la liste "Rendering System Options" suivant vos préférence
Clicker sur OK

Notice d'utilisation

Déplacement de la caméra
Q= gauche
S= arriére
D= droite
Z= avant
A= vers le haut
E= vers le bas
bouger la souris pour changer l'orientation de la caméra (attention la souris est très sensible)

Déplacement du spot Bleu
J= gauche
K= arriére
L= droite
I= avant

Ajouter/ Supprimer des personnages
F=supprimer un ninja
R=ajouter un ninja
Y=supprimer un robot
H=ajouter un robot

Modification du temps
W = accélération de la simulation
X = ralentissement de la simulation

Arrêt de la simulation
Alt+F4

Bug connu
Lors de la création de personnage ou lors de l'utilisation il se peut qu'un personnage se mette à marcher dans le vide, il s'en suit un plantage de l'application
